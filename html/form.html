表单验证

验证使用说明：
在input上面加一个 data-validate属性
传入一个JSON对象
{
	validate : { //验证方式:提示
			notempty : '不能为空！',
			email : '请输入正确的邮箱地址'
		}, 
	style : 'fadeRight' //目前只有 default 和 fadeRight 两种显示方式
}

//获取当前mt-form中的出错个数，如果没有错，返回0
调用方法：$(".mt-form").mtValidate();

//清除所有错误
调用方法：$(".mt-form").mtClearValidate();

//清空表单,重置
调用方法：$(".mt-form").mtClearForm();

validate的值 

decmal: 浮点数
decmal1: 正浮点数
decmal2: 负浮点数
decmal3: 浮点数
decmal4: 非负浮点数（正浮点数 + 0）
decmal5: 非正浮点数（负浮点数 +0
intege: 整数
intege1: 正整数
intege2: 负整数
num: 数字
num1: 正数（正整数 + 0）
num2: 负数（负整数 + 0）
ascii: 仅ACSII字符
chinese: 仅中文
color: 颜色
date: 日期
email: 邮件
idcard: 身份证
ip4: ip地址
letter: 字母
letter_l: 小写字母
letter_u: 大写字母
mobile: 手机
notempty: 非空
password: 密码(数字，字母，下划线组成)
fullNumber: 数字
picture: 图片
qq:  QQ号码
rar: 压缩文件
tel: 电话号码的函数(包括验证国内区号,国际区号,分机号)
url: url
username: 户名
deptname: 单位名
zipcode: 邮编
realname: 真实姓名（中文）
companyname: 公司名称（中文）
companyaddr: 公司地址
companysite: 公司网站
Example: 
{
	validate : { //验证方式:提示
			notempty : '不能为空！',
			email : '请输入正确的邮箱地址'
		}, 
	style : 'fadeRight' //目前只有 default 和 fadeRight 两种显示方式
}
